# PescApp

## Funcionalidades
- Registro e inicio de sesión de usuarios
- Creación de reportes de pesca
- Adjuntar imagen al reporte
- Listado y detalle de reportes
- Filtro de reportes por fecha
- Visualización de datos detallados

## Tecnologías utilizadas
- Lenguaje: Kotlin
- Interfáz de Usuario: Jetpack Compose + Material 3
- Arquitectura: MVVM (ViewModel, View, Model)
- Base de Datos Local: Room (SQLite)
- Base de Datos en la Nube: Firebase Firestore
- Autenticación: Firebase Authentication

## Requisitos previos
- Android Studio Giraffe o superior
- JDK 17
- Android SDK (API 24 o superior)
- Cuenta de Firebase

## Configuración del proyecto
1- Clonar el repositorio
git clone https://github.com/FrancoDeMarco/ProyectoPesca2025
2- Abrir el proyecto en Android Studio
- Abrir Android Studio
- Seleccionar Open an existing project
- Elegir la carpeta del proyecto
- Esperar a que Gradle finalice la sincronización

## Instrucciones de compilación y ejecución
1- Seleccionar un emulador o celular físico.
2- Presionar "Run".
3- La app se compilará e instalará
